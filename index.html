Desarrollo web responsivo

Su sitio web será responsivo, eso significa que su sitio web cambiará de tamaño de manera fluida para una visualización óptima, independientemente del tamaño de la pantalla o del dispositivo (por ejemplo: Smart TV, computadora, iPhone, iPad).

Administración de páginas web

Con los serivicios de web master te brindaré la atención y dedicación a tu página web, otros servicios relacionados como: diseño web, desarrollo de la página, aspectos de imagen, posicionamiento, seguridad web, reportes de navegación y actualizaciones que se requieran.

Tecnologías

Tecnologías web que utilizó

En la actualidad utilizó tecnologías Frontend para el desarrollo de aplicaciones web, como también me encuentro en conocimiento y practicas constantes para dominar tecnologías Backend.

HTML5

Desarrollo Web Frontend

Un lenguaje de hipertexto de etiquetas que se encuentra en continua evolución, siendo el más actualizado el HTML5.

CSS3

Desarrollo Web Frontend

Se trata de una tecnología utilizada para dotar de cualidades visuales y estéticas a una página web. Siendo el más actualizado CSS3.

JavaScript

Desarrollo Web Frontend

Es el encargado de aportar interactividad al desarrollo, programa el comportamiento de los elementos aportando dinamismo a la interfaz.

PHP

Desarrollo Web Backend

Es una de las mejores tecnologías para desarrollo web. Se usa para conectar la plataforma web con el servidor de datos.

MySQL

Gestor de Base de Datos

Se utiliza para almacenar la información, y trabaja con múltiples tablas que permiten crear nuevas bases de datos y hacer cualquier consulta.

Tableau

Software de Análisis de datos

Tableau es una herramienta de visualización de datos potente utilizada en el área de la Inteligencia de negocios.
